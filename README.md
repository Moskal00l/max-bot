# max-bot
Бот и сайт в нашем проекте запущен в облаке Yandex.Cloud, во время проверки Вам не потребуется запускать локально. Но если Вам очень принципально, мы оставим ниже инструкцию по созданию Docker-образа бота и его запуска локально.
P. S. Во избежание двойных обработок, перед локальным запуском нашего бота, Вам потребуется отключить подписку, с чем мы Вам поможем.

Команда для создания Docker-образа:
docker build -t max-bot:latest .

Команда для запуска Docker-контейнера:
docker run -e TOKEN='ваш токен бота' -d moskal01/max-bot:latest
